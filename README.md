# node-cli-boilerplate

## Supports

- ES6 (using `babel`)
- Auto testing (using `jest`)

## Start a project

Make sure you have installed [Node.js](https://nodejs.org/).

1. `git clone` this repo
2. `cd` to this repo dir, and run `node setup`
3. Happy hacking
